# JavaScript Quiz App

A simple and interactive **Quizweb** built using **JavaScript**, focusing on **DOM manipulation, event listeners, and CSS Flexbox**.

This project contains **5 multiple-choice questions**, where each question has **4 options with only one correct answer**.

---

##  Features

- Start Quiz button
- 5 multiple-choice questions
- 4 options per question (single correct)
- Score tracking
- Restart Quiz functionality
- Interactive UI
- Responsive layout using CSS Flexbox

---

##  Tech Stack

- HTML
- CSS (Flexbox)
- JavaScript (Vanilla JS)

---

## How the Quiz Works

1. User clicks Start Quiz
2. Questions appear one by one
3. User selects one option
4. Answer is checked
5. Score is updated
6. After last question, result is shown
7. User can click Restart Quiz to play again
